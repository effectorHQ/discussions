# OpenClawHQ Discussions

This repository hosts **Discussions** for the [OpenClawHQ](https://github.com/OpenClawHQ) organization.

- **No code lives here** — it's used only for GitHub Discussions (community chat, ideas, Q&A).
- **Org-level discussions** are linked from [OpenClawHQ/.github](https://github.com/OpenClawHQ/.github) so they appear in the org’s Discussions tab.

## How to use

- Go to [**Discussions**](https://github.com/OpenClawHQ/discussions/discussions) to browse or start a conversation.
- New here? See the [org README](https://github.com/OpenClawHQ/.github) and [CONTRIBUTING](https://github.com/OpenClawHQ/.github/blob/main/CONTRIBUTING.md).

## Optional: discussion templates

Templates live in [`.github/DISCUSSION_TEMPLATE/`](.github/DISCUSSION_TEMPLATE/). Each file name must match a **category slug** (e.g. `general.yml` for the “General” category). If you add or rename categories in repo Settings → Discussions, add or rename the corresponding `.yml` file.
